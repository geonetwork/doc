# GeoNetwork Documentation

For information about the GeoNetwork project please visit:

* https://docs.geonetwork-opensource.org/latest/
* https://docs.geonetwork-opensource.org/stable/

To contribute:

* [wiki](https://github.com/geonetwork/doc/wiki)   
* [Documentation Guide](https://docs.geonetwork-opensource.org/latest/devel/docs/)

Introduce yourself on the [GeoNetwork User Forum](https://discourse.osgeo.org/c/geonetwork/user/54) to request wiki access. We can also coordinate documentation updates and translations.
